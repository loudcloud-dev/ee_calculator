# README

## Requirements
- Ruby >= v3.3.5
- Rails >= v7.2.1

## Installation 
```bash
$ git clone https://github.com/loudcloud-dev/ee_calculator.git
```
```bash
$ cd ee_calculator
```
```bash
$ gem install bundler
```
```bash
$ bundle install
```

## Database Setup
```bash
$ rails db:migrate
```
```bash
$ rails db:seed
```

## Run the app
```bash
$ bin/dev
```
